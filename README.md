# Tantan's Hangout Mod.io Map Template
This is a template unity project for custom map via mod.io on Tantan's Hangout.

# Requierments
* Unity Version 2022.3.32ft

# Template Tutorial


## Main Rules
* Do NOT remove, scale, or move the "(DO NOT DELETE ME)" gameobject in the scene.

## Exporting 

1. Select Tools> VRMapModExporterWindow
2. Fill out all of the shit
3. Select export
4. You should have had a file explorer window pop up after exportation, open the folder that is highlighted, select all contents in the file/folder the window was opened in and zip it.

## Upload To Mod.io

1. Go to https://mod.io/g/tantans-hangout?preview=869242ce0e17c3c7295b5d0c9bfbb5fb
2. Click "Add Map" in the top left corner on the page
3. Fill out the Title, Summery, Map Listing Image, Description, and Tags.
4. Select "Create Map" at the bottom of the page.
5. Hover above your map listing and select the pencil/pen on the top right corner of your listing.
6. On the left side, select the "Files" option.
7. Select "Add a New File", and then select the .zip file you made earlier.
8. Fill out all of the blanks (bedsides Metadata blob), select the check to "I agree to the terms and privacy policy", and then select "Upload And Publish"
