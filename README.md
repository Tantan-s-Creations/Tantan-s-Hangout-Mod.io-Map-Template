# Tantan-s-Hangout-Mod.io-Map-Template
This is a template unity project for custom map via mod.io on Tantan's Hangout.

# Requierments
* Unity Version 2022.3.32ft

# Template Tut


## Main Rules
* Do NOT remove, scale, or move the "floor" in the scene

## Exporting 

1. Select Tools> VRMapModExporterWindow
2. Fill out all of the shit
3. Select export
4. You should have had a file explorer window pop up, select all contents in the file/folder the window was opened in and zip it.
